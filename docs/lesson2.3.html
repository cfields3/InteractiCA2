Cave Generator

Some video games use cellular automata to create cave-like levels. Experiment to find this ruleset.

This pattern only needs two of the four rules. Let the equality statement for one rule on each side to X.

The only active rule for cell birth should be 'neighbors > 4' and the cell death rule should be 'neighbors < X'. X is the same for both. Use the 'Random State' button if the board is blank.
